Плагин главного табло. В основе дизайна HTML.

Остальные файлы в папку config сервера.

Привязка к рабочим местам

id - Абстрактный идентификатор рабочего места. Привязывается к номеру пункта приема посетителей в bs.adr. Пункт приема посетителей это атрибут пользователя "Идентификатор"
[id|blink_N]  пометить миганием, N - просто абстрактный счетчик. Это значение параметра id для тега в html
[id|name] text название услуги вызванного посетителя
[id|discription] text описание услуги  вызванного посетителя
[id|user] text название пользователя  вызвавшего посетителя
[id|ext] text дополнительная колонка пользователя  вызвавшего посетителя
[id|point] text атрибут пользователя "Идентификатор"
[id|{N}] A123 номерок посетителя. N - порядковое положение посетителя в очереди к услуге id


Привязка к строкам вызова

nStr - номер строки
{nStr|blinkR_} - Это значение параметра id для тега в html. Строки могут мигать при вызове.
{nStr|N} - будет заменено значением номера тилона в строке nStr 
{nStr|point} - будет заменено значением идентификатора пункта приема в строке nStr
{nStr|ext} - будет заменено значением расширенного стролбца для юзера в строке nStr

пример:
Пятая строка табло вызова
<tr>
<td width="50%" align="center"><span style='font-size:30.0pt;color:yellow' id="{5|blinkR_5}">{5|N}</span></td>
<td width="50%" align="center"><span style='font-size:30.0pt;color:yellow' id="{5|blinkR_5}">{5|point}</span></td>
<td width="50%" align="center"><span style='font-size:30.0pt;color:yellow' id="{5|blinkR_5}">{5|ext}</span></td>
</tr>
